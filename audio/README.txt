Recommended workflow:
1. Generate 2-5 minute instrumental clips in Suno, Udio, or another tool.
2. Keep prompts short and specific:
   "instrumental, deep synth focus music, 90-102 bpm, warm sub bass, muted kick, soft analog pads, sparse delayed melody, no vocals"
3. Curate only the strongest clips.
4. Export them as `.mp3` files with the names above.
5. Start the Astro app and the player will shuffle them.
